Python Hotel Booking System

In this project, my group and I worked on an engaging project: building a Hotel Booking System using Python. The Hotel Booking System was designed to simplify booking processes for customers. It allowed them to check available rooms, make reservations, and manage their bookings effortlessly. Behind the scenes, the system handled room availability, pricing, and customer details smoothly. It also generated invoices for customers upon booking confirmation. Throughout this project, we put our Python skills to work, learning how to use loops, functions, arrays, and file handling effectively.

JMB Send Monthly Invoice System (G Suite Scripting)

"It is one of the project that I developed during my internship program. As an IT Support, my task it to create automation systems to help the company. One of my task it to create Send Invoice System. This system was created purely using G Suite Scripting which includes, Google Form, Google Sheets, Google Apps Script as well as Zapier automation. The invoice will send to the owner according to the owner email using (Gmail service) and phone number using WhatsApp (Watoolbox automation) provided in the Google Sheets. When admin want to send the invoice, the form will display the unit number so that admin do not lose track. The unit number is also traced in the system. After a full cycle, all the log records will be sent to the boss for record."
